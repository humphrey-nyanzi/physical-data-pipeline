# Reports

This directory contains generated analysis reports for clubs in both leagues.

## Structure

- **FWSL/** - Reports for FWSL (Uganda Women's Football Super League)
- **UPL/** - Reports for UPL (Uganda Premier League)

## Report Contents

Each report (Word document) includes:

- **Team Summary** - Season overview and key statistics
- **Player Profiles** - Individual player performance metrics
- **Performance Metrics**
  - Distance covered per match
  - High-speed running
  - Sprinting activity
  - Acceleration/deceleration events
  - Player load (intensity)
- **Trends & Insights** - Time-series trends, comparisons
- **Charts & Visualizations** - Embedded PNG charts

## Updating Reports

1. Update raw data in `data/raw/`
2. Run `01_clean_data.ipynb` to process
3. Run `03_generate_reports.ipynb` to regenerate Word reports

Reports are overwritten with each run.
